## Root Cause

The vulnerability is a memory leak (CWE-401) in `glutAddSubMenu` at line 276. Memory is allocated for `menuEntry` via `calloc`, but when `fgState.ActiveMenus` is non-zero (indicating menus are currently in use), the function calls `fgError()` at line 281, which invokes `fgDeinitialize()` and then `exit(1)` at lines 263-266. Since `exit()` terminates the program immediately, the allocated memory for `menuEntry` is never freed, causing a missing release of memory after effective lifetime.
